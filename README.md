# Console Chart

> 阿里云控制台业务图表库

旨在提供一款**开箱即用、统一视觉方案**的图表组件库。

## Install

```bash
yarn add @alicloud/console-chart --save
```

## Usage

```js

// 引入组件
import { ConsoleLineChart  } from '@alicloud/console-line-chart';

<ConsoleLineChart data={ ...data } config={config} />
```
